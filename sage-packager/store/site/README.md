# SAGE Privacy Policy — Website

A ready-to-host privacy policy site for **SAGE (smart interactive reminder)** by Network Creation LLC.

## Files
- `index.html` — the page (references local images in `assets/`)
- `assets/company-logo.png` — Network Creation logo
- `assets/app-name.png` — SAGE app-name banner
- `privacy-hosted.html` — standalone version that pulls images from public URLs (no assets folder needed)

## Fastest ways to publish (pick one, all free)

### GitHub Pages
1. Create a public repo, upload `index.html` + the `assets/` folder.
2. Settings ▸ Pages ▸ Deploy from branch ▸ `main` / root.
3. Your URL: `https://<username>.github.io/<repo>/`

### Netlify Drop
1. Go to https://app.netlify.com/drop
2. Drag the whole `site/` folder in.
3. Copy the generated URL.

### Cloudflare Pages / Vercel
Point it at a repo containing this folder — no build step needed (static site).

## Use the URL
Paste the final public URL into:
- App Store Connect ▸ App Privacy ▸ Privacy Policy URL
- Google Play Console ▸ Store listing ▸ Privacy Policy
- `store/STORE_LISTING.md`

Contact email in the policy: **network.creation@outlook.com**
